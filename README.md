# chainviz

ブロックチェーンシステムの内部（ノード・プロセス・通信）をリアルタイムに可視化する
インタラクティブキャンバスアプリ。学習・開発理解のためのプロジェクト。

現在は構想段階。全体像は [docs/CONCEPT.md](docs/CONCEPT.md) を参照。

## コンセプト

- Docker 上に Ethereum ノード群を起動し、キャンバス上にカードとして表示
- Miro のようにズーム・パン・ドラッグで操作
- ホバーで IP・内部プロセス・同期状態を表示、ピア接続やチェーン所属をエッジで表現
- インフラ層 → P2P 層 → トランザクション層 → ノード内部層 とレイヤーを切り替えて観察
